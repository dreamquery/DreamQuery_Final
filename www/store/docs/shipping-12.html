Shipping - Quantity Rates

Overview
Quantity based shipping is a calculation based on the amount of tangible goods in your cart. Quantity based options can be specified per zone and can be a set amount or a percentage of the tangible goods total.
Select Zones
Specify which zones this new rate applies to.
Specify Quantity Range
Specify item count range for this rate. The amount of goods between this range will have the shipping applied. Digits ONLY.
Quantity Based Rate
Specify amount to apply if cost of goods is between specified range. Can be set cost or percentage of goods total. For set cost, enter cost with no commas, eg: 9.99. For percentage enter percentage amount, eg: 25%
Enable Rate
Enable or disable this rate
Batch Enable/Disable
Click the to batch enable or disable rates. Click the link and select the zones applicable.
Manage Rates
Use the links provided to edit or delete rates. Use the icon to filter by zone.
