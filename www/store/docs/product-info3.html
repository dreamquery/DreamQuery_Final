Product - Copy Attributes

Overview
This feature lets you batch copy attributes from one product to one or more other products. Useful if you have similar products that have similar attributes.
Copy Attribute(s) to Selected Product(s):
Attributes are shown for the current product. Use the checkboxes to select which attributes you want to copy.
Copy To
Select a category to load products from that category and use the checkboxes to select which products you wish to copy attributes to. Keep making selections and clicking 'Copy Attributes' until you are happy the relevant products have been updated.
Product Quick Links
To access other product areas, click the button.
Cancel
Click to return to main products screen.
