Shipping Countries

Overview
Select the countries that are applicable for shipping. Further filters are available on various pages, but any countries not enabled and not shown anywhere else.
Enable Countries
Use the checkboxes to select which countries you ship to.
Add/Edit Countries
You can add or edit any countries not in the list via the boxes provided. Click any country name to edit.

Country Name - Enter name of country
3 Char ISO Code - 3 character country ISO3166 code
2 Char ISO Code - 2 character country ISO3166 code
ISO4217 Code - 3 digit country ISO4217 code
Enable Country - Enable or diable country

More information on currencies and codes:
http://en.wikipedia.org/wiki/ISO_3166-1
http://en.wikipedia.org/wiki/ISO_4217

ALL fields are required for any country. Country codes are required for certain payment gateways and must be correct.
