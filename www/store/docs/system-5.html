Latest News Ticker
Maian Cart utilises the excellent jQuery marquee plugin created by Aamir Afridi.
Add New Ticker
Enter news text. HTML may be used here if required, but be careful as some HTML may break the ticker. Hyperlinks only are recommended as follows:

Check out our store bargains <a href="http://www.example.com/cart">here</a>

In this example, 'here' would hyperlink to 'http://www.example.com/cart'. You can use full, relative or external links.
Enable Ticker
Do you wish to enable this news ticker text? Yes or No.
Ticker Page Load
By default the ticker only loads on the homepage. If you wish to load it on all pages, edit the following in the 'control/defined.php' file:

define('NEWS_TICKER_DISPLAY_PREF', 1);
Change to 0 to load on all pages.
Drag and Drop to Order
To set the order that the ticker messages display, drag and drop the rows after creating news tickers. A move icon appears if you hover your cursor over the rows. Simply drag and drop to re-order.
