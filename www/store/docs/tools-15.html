Send Mail - Newsletter/Mailer

Overview
Here you can batch mail accounts. You can save layouts as templates for easy recall later.
Top Bar Options
- Saves current template set. If it already exists, it will be updated.

- Search templates. As you start to type suggestions will be shown. Select template to load.

- Click to edit templates
From Name, Email & Subject
Enter from name, email and subject. Use variables (between braces) if applicable to personalise.
HTML / Plain Text Message
You should enter an HTML and plain text version of your message. Many people don`t allow HTML.

Use variables (between braces) if applicable to personalise. You should ALWAYS include the subscribe option "{unsubscribe}". With the HTML message it must be as a hyperlink as follows:

Click <a href="{unsubscribe}">here</a> to unsubscribe.
Note: For bigger textareas, click the crosshairs, top right, in the textarea boxes.

Clicking the 'Preview' option for HTM loads a preview window in an iframe.
Attachments
You can include multiple attachments, but large attachments are not recommended as the system needs to process the files before they can be used. More robust software should be used for large attachments. Use the buttons to add / remove attachments.
