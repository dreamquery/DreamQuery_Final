Mail Accounts

Overview
Batch mail accounts. Note that this is a basic mailer. If you have many accounts, you may prefer to export and import the accounts into more robust software, better suited for batch mailing.
Subject / Message
Enter subject and message. Each can be personalised using:

{name} = Name
{email} = Email

Note: For bigger textarea, click the crosshair, top right, in the textarea box.
Account Type
Send to all accounts, personal ONLY or trade ONLY. Total enabled / verified accounts counts are shown.
