Add Sale

Overview
Here you can manually add a sale to the database. This is useful if for example, you take an order by phone and want to add it to the system.
Add Products to Sale
Use the tabs provided to add products to the sale. Click the "Add Products to Sale" link for each to locate the add screens to select products. Once you are on those screens click the help link again for more info on those pages. Once the add page has reloaded, specify the product quantity for each product.
Product Attributes
When the add page is reload, an "Add Attribute" link will be shown to load attributes if required. Clicking the link will at first show a drop down menu of available attributes.


Select an option to reload. If additional cost is applicable it will be shown. You can enter a custom value if you wish.


Click the icon in price box to clear the attribute.

Once you have added the attributes, click the icon to refresh the prices. A running total of attributes cost will be shown in the right menu beneath the cost of the product multiplied by the quantity.

Personalisation
When the add page is reload, if personalisation is available, boxes will be shown for you to enter the information. If additional cost is applicable it will be shown. You can enter a custom value if you wish.

Payment/Shipping Overview
Select or enter the information relevant to this sale. When you select shipping selectors from the drop down menu the totals will auto appear in the boxes. Note that a minus symbol indicates a deduction field, like a discount and a plus symbol indicates an addition. Use the "Recalculate Totals" option to total up everything before continuing. This is located via a refresh icon in the grand total box.


Make sure you have clicked the icon first to update the product prices. This is mainly related to attributes and personalisation. See note above.
Account
Enter name and email for buyer. When you start typing in the name or email field suggestions will be shown for existing accounts. Select an account to load or enter new details.

If An Account Doesn`t Exist, Create It - Check box to create account if it doesn`t exist.
Send Email Notification For Account Creation - If account is being created, check box to send email notification to buyer.
Billing & Shipping Address
Enter billing and shipping address. Use the icon to copy field data from billing to shipping and vice versa. If you have selected an existing account via the auto complete, the address fields are auto filled in.
Sale Options
IP Address(es) (Separate with comma) - Enter IP address of buyer if known. Auto completed by system if blank when buyer accesses order page.
Purchase Date - Enter purchase date
Purchase Time (HH:MM:SS) - Enter purchase time. Format MUST be HH:MM:SS
Purchase/Payment ID - If applicable, enter purchase or payment ID
Invoice No - Use the icon to calculate next invoice number or enter manually. Note that next invoice will continue from the number entered.
Sale Notes
For backend admin notes, not seen by buyer.

Note: For bigger textarea, click the crosshair, top right, in the textarea box.
Edit Status & Notes
Do you want to write order status? It is recommended you write statuses to keep track of order operations, but it is optional.

Note: For bigger textarea, click the crosshair, top right, in the textarea box.
Clear & Reload
Clears all data and reloads page with blanks. Only click if you want to cancel and start again.
