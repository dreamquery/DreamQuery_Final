Maian Cart Overview - Language Switcher

Language Switcher
This enables visitors to see the base language data in their own language. At the moment product & category data isn`t supported.

Please see the language section for enabling additional languages.
Language Selection
Selection is made via the drop down menu at the top of the screen. This shows language options. Click button again to close with no selection. The current selected language is shown with a icon. Select language to reload screen. See following preview:


If a visitor is logged in, language setting is remembered for next time.
