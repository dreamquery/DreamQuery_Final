Version Check

Overview
The version check connects to the Maian Script World servers to check the version number against the version number you are currently using.

DISCLAIMER: NO personal identifying information is sent for this option. The ONLY thing sent is the version number from your database and the only thing returned is the current version number. NOTHING is logged by Maian Script World and this check is anonymous.
This option is only shown to none restricted admin users.
Manual
If you prefer to get the current version number manually, simply access the following in your browser:

https://www.maianscriptworld.co.uk/version-check.php?id=11
Disable
The version check link can be disabled in the "admin/control/defined.php file.

define('DISPLAY_SOFTWARE_VERSION_CHECK', 0);
