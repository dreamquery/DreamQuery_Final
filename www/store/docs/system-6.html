Theme Switcher

Overview
The theme switcher lets you change the front end theme on any given date or date range. This is useful if you want to load different store themes at different times. Maybe holiday seasons like Christmas or during a special offer promotion. If no switch is found, the default theme is loaded.

Note that if you specify dates that overlap, the first theme found will be the one that loads. Also, if a theme is specified for a category, this will take priority and any themes in given date ranges are ignored when the category page loads.
From - To Date
The from and to date for the theme switch. For a single date, set the same to and from date. Both dates must be set.
Theme to Load
The theme to load. This must be a folder prefixed "_theme" in the "contents" folder. More info here.

If the theme isn`t found, the default theme is loaded as set in your settings.

The default theme won`t be shown in the list. If no other themes are available the switcher can`t be enabled.
Enable Switch
Enable theme switch, yes or no. This option lets you disable a theme load without deleting it.
