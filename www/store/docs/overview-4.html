Maian Cart Overview - Search

Search
The search option enables visitors to find products in your store. A basic search box is shown at the top of the screen:


To search, enter keyword (single or space delimited) then enter return or click the icon. This isn`t shown on small screens.
Advanced Search
Advanced Search is accessed by clicking the icon to the left of the search box. This screen offers a more detailed search as shown below:

Price Range
Narrow search down to given price range via price slider. Drag the sliders to alter the range. See preview:

Filters / Sort By
After a search has been performed it can be filtered by the options in the left menu:


Sorting is also available via the sort drop down at the top of the page:

Saving Searches
After a search has been performed it can be saved for later recall. A visitor MUST have an account to save searches. To save search, click the link in the left menu and enter a name / reference. If the visitor isn`t logged in an error will appear saying they must have an account.
