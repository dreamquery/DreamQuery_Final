Disabled / Unverified Accounts

Overview
If you disable an account, it will move to this screen. Other accounts show here are accounts which for some reason were never verified. Maybe someone signed up for an account and never got the verification email etc.
Top Bar Options
- Click to add new account

- Click to filter list view options

- Click to view search box to search accounts
Account Specific Options
- Click to edit account

- Click to add / edit backend notes

- Click to expand area for more options (below).
Expanded Area Options
Displays basic account information only for this page.
Delete
Use the checkboxes to delete accounts.
Resend Verification Email
This option ONLY appears when there are only unverified accounts on the screen or if you filter by unverified acounts. You`ll see the following button.


Use the checkboxes provided to highlight the accounts you wish to resend verification for. You can also enable the account manually via the edit screen.
