Overview
Manually add a new buyer account to the database. Accounts can be personal or trade. More info below.

Useful for security. Logins viewable on the entry log page.
Addresses / Notes
Enter default shipping addresses and backend notes. Use the icon to copy addresses between shipping and billing and vice versa. Notes for backend only, not seen in account area.
Marketing Tracking Code
Only shown on edit screen and only if marketing trackers exist. This is the tracking code assigned to this account (if applicable) via a tracking code url. More info on trackers here.
Account Type
Standard or Trade. Trade accounts can have global discounts set. Useful if you sell to the trade industry. If the trade option is set, other options will appear as shown below. NOTE that the following store links won`t be seen by trade logins:

Special Offers Screen
Gift Certificates
Wish Lists

Advanced Search Price Points & Filters

NOTE: For trade logins, stock does not reduce on products / attributes after a purchase occurs.
Trade ONLY - Trade Discount (%):
This is a global trade discount across all products. This discount overrides all other discounts when this person is logged in. So, if a global store discount and product special offers etc are set, these are ignored in favour of the trade discount. Set discount in percentage value ONLY.
Trade ONLY - Trade Min Quantity Override
If set, trade buyer must always add the min quantity to their basket. This overrides any regular product min restriction.
Trade ONLY - Trade Max Quantity Override
If set, trade buyer can buy no more than the max allowed products. This overrides any regular product max restriction.
Trade ONLY - Min Checkout Amount
If set, goods total must be greater than or equal to min checkout amount before buyer can complete checkout procedure. Leave blank for no restriction.
Trade ONLY - Trade Product Stock / Attribute Override
Overrides stock and attribute values globally across store, but only for trade account. Example, for regular visitors / accounts you may have a stock of 5 for any given product, but an override of 100 here, would be seen as stock 100 for trade accounts.

Note: If a product is shown as out of stock for a standard visitor, setting an override here makes all products available for trade logins. To hide for trade logins, disable product in admin control panel.
Send Email Notification
If set, sends email notification to account email with log in details.
