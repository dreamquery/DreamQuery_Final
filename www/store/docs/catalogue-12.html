Download Manager

Overview
The download manager enables you to easily manage your downloadable products via a javascript file manager powered by elFinder. This is experimental in Maian Cart, so if you find the download manager not suitable or you can`t get it to work, use an FTP client such as FileZilla or WinSCP.

Downloads should be stored outside of your web root for security. This means NOT accessible in a browser. The file manager may not be fully compatible with files outside the root, so in this case, use FTP as mentioned above.

A few basic operations are shown below.
Locale
English is the default language for the file manager. To change locale, update 2 letter code in the 'admin/control/defined.php' file.

define('ELF_LOCALE', 'en');
File must exist in the 'admin/templates/js/i18n' directory.
Files & Folders

NOTE: It is highly recommended and good practice that files and folder names do NOT contain spaces or invalid characters. Use hyphens (-), underscores (_) or alphanumeric characters (a-z,0-9) to be safe.
