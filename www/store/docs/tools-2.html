Update Stock Levels

Overview
This feature lets you batch update your stock levels for products and attributes.
Stock Adjustment
Specify stock adjustment.
Omit Where Product/Attributes Prices are Between
Do you wish to omit stock levels between a certain range? Both ranges must be set.
Apply to Products Added ONLY Between Selected Dates
Do you only want to apply the update to products that were added between a certain date? YYYY-MM-DD format ONLY. Both dates must be specified.
Adjustment Type
Do you wish to decrease or increase the stock with the new amount or add a new fixed amount?
Apply Adjustment To
Do you want this update to apply to attributes and products or both? If none are specified, defaults to products.
Select Products NOT to update
If applicable, specify products NOT to update. Use the radio buttons and categories provided.
