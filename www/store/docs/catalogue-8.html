Special Offers

Overview
Enables you to batch add special offers to the system. Alternatively, you can add any product as a special offer by simple adding an offer price on the add or edit screens of any product.
Discount Rate
The special offer batch page enables you to add products as special offers and specify fixed offer amounts or amounts based on a percentage of the original cost. Enter fixed amount or percentage in the 'Discount Rate' box. For example, 5% would discount the products original total by 5%. If you entered 1.99, this would discount the products original total by 1.99.

Special offers are marked in the store as follows:


Styling can be adjusted via the stylesheets.
Expiry Date
An expiry date can be set to auto expire the special offer and have the product return to its original price.
Multi Buy Only
For multi buy, enter amount to be purchased before discount applies. Set to 0 to disable multi buy.
Omitting Products
Use the category filter and checkboxes next to each product in a list to NOT have the selected products updated.
Manage Offers
Click the icon to filter offers by category. Clicking the icon will clear ALL offers. Use the edit and delete options to edit or delete current offers.
Offers in Multiple Categories
NOTE that if you have products in multiple categories, these offers will be shown in each category. Clearing an offer for a product will clear it from ALL categories it is related to.
