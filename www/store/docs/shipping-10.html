Tare Weight

Overview
Tare weight, sometimes called 'unladen weight', is the weight of an empty vehicle or container. For example, lets say you have a product that weighs 4KG, but when packaged the weight increases to 5KG, the tare weight would be 1KG. Sometimes its not viable to simply increase the freight costs to cover tare weight, instead a small surcharge is preferred, which is added to the shipping cost to cover cost of packaging etc.

This feature is ONLY used in conjunction with weight based rates and is optional.
Weight From / Weight To
Weight range. To specify only a single rate, enter a high amount for the 'from' field. This can be any kind of weight band, but must match your product weight. For example, if your products are in grammes, use a gramme range. eg: 0 to 1000. If pounds and ounces you might start 0 to 5.
Tare Weight Cost
Tare weight cost for this weight band. This can be either a fixed amount or a percentage. If percentage, it will be the percentage of the shipping calculation.

Example 1: Shipping calculation for products in basket comes to £4.99. Tare weight for this weight band is 0.50p, so new shipping rate is £5.49.
Example 2: Shipping calculation for products in basket comes to £4.99. Tare weight for this weight band is 5%, so new shipping rate is £5.24.

If shipping calculation is 0.00, but a tare weight is applicable, the tare weight will still be added to the shipping total.
Apply to Service
Which service(s) does this rate apply to? Use the checkboxes to batch add rates to multiple services if in add mode. If in edit mode, use select option.
Batch Adding Rates via CSV
Use the locate CSV option to batch add tare rates. To batch add, csv must be a comma delimited list in the following format:

Weight from, Weight to, Tare Weight Cost (no commas, etc, 9.99)

Example:

0, 10000, 1.99
100001, 500000, 5.99

For batch operation, file and checkbox selection of applicable services is required.
