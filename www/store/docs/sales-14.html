View Sales - Manage Status Text

Overview
This screen lets you delete or edit existing order update statuses. On initial load you`ll see a drop down menu of status references. Click to view details.
Ref / Subject / Text
Enter reference, title and text.

Note: For bigger textarea, click the crosshair, top right, in the textarea box.

If a reference is chosen that already exists an error will display. HTML should not be used unless you have set up HTML mail parsing tags.
Update / Delete
Use the buttons at the bottom of the screen to update or delete a status.
