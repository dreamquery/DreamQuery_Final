Store Banners

Store Banners
A useful promotional feature is the store banner rotator. You can add as many banners as you like to the system and each banner can have an optional url to link to an internal or external site link and be category specific. This can be useful if you are running different promotions during the year. Banners can also be easily enabled/disabled after being added.

NOTE: Due to screen limitations and to save space, banners are NOT shown on mobile devices.
Banner Caption
Banner caption appears below the image. Leave blank for no caption.
Banner Url
Url is optional. If specified, banner is clickable. This can be an internal or external site url. Examples:

http://www.example.com (External to another site)

?pd=4 (Internal to store product page)

?c=9 (Internal to store category page)

http://www.example.com/cart/?pd=4 (Also internal to store product page)
Banner Image
Locate an image file on your hard drive. Images should be a valid image format like jpg,jpeg,png,gif.

Image will display full screen width, so bear this in mind. An optimal size for banner images for full screen is around 2000px 550px. Images should be optimised for the web and not for example, uploaded directly from a digital source (eg: camera or phone) as the file size will be too big. This impacts on bandwidth and load time.

If you wish to NOT rename banners or change the banner prefix, edit the 'admin/control/defined.php' file.

define('BANNER_PREFIX', 'customprefix_');
define('RENAME_BANNERS', 0);
Banner Categories
If categories are specified, banner image will load only when a category is being viewed or when a product is being viewed. If none category specific, loads on all pages.
Display on Store Main Page Only
If set, banner displays on store main page only and no other screens.
Display Between Dates
Use this option if you only want the banner image to display between certain dates.
Display for Trade Accounts ONLY
If set, displays only for trade account logins.
Enable Banner
Do you wish to enable or disable this banner?
Store Banner Image Slider Options
The Maian Cart banner rotator is powered by Slippry a simple image slider written by Lukas Jakob Hafner & Thomas Hurd. To change the settings, edit the following file:

content/*theme*/html/js/banner-slider.htm
Banner Ordering
To change the order the banners appear, hover your cursor over the banner rows until you see a crosshair. Click and drag to re-order.
