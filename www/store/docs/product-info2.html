Product - Attributes & Attribute Groups

Overview
Product attributes are items related to the main product. Might be different sizes for clothes or different colour options. You can also batch add attributes.
Attribute Groups
Attributes are listed together in groups. For example, if you are selling laptop computers, you might have the following attribute structure:

Screen Size
12"
15"
17.3"
Operating System
Vista
Win7
WinXP
Extras
Carry Case
Logitech Mouse
Transcend 8GB Flash Drive
Visitors can select multiple attributes when adding item to basket. One from each group if multiple option is specified or else its one per group.
Adding Attributes - Group Information
Add to Existing Group - If you wish product attributes to be added to exiting group, select it from the drop down.

OR Enter New Group Name - Enter new group name if applicable.

Allow Multiple Selections - If set, allows visitor to choose more than one option when adding product to basket.

Required Selection - If set, requires at least one selection from visitor when adding product to basket.
Adding Attributes - Product Attributes
Name - Enter name for attribute. Max 100 chars.

Add. Cost - Enter additional cost if applicable.

Add. Weight - Enter additional weight if applicable. Will affect weight based rates.

Stock - Enter stock level. If unlimited, enter a high number.

Order - Specify order display preference.

Click the buttons to add multiple rows. Click the "Add New Attributes" button to add atributes.
Manage Product Attributes
You can change the display order for attribute groups by dragging and dropping the rows. Click the icon to edit a group and the icon to delete a group. If a sale exists that contains items from an attribute group, you will be prompted and the group cannot be removed.
