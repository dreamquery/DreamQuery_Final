Maian Cart Overview - Categories

Categories
Categories are displayed via the 'Categories' button, top left of the screen. Once clicked, the screen slides to the right to reveal the categories. Clicking arrows inside the menu reveals child and infant categories if enabled. See below animation:

Category List Product View
Products are shown in a list, either in a grid view or list view. The default preference can be set in the settings. Previews for both views are as follows:



The wish list button is only shown if the wish list is enabled. In the grid view, buttons only show icons. View can be switched from grid view to list view and vice versa by using the buttons at the top of the listing. If a visitor is logged in, the selection will be remembered

Sale Items
Items on sale are shown with a sale flag. Expiry dates are also shown with a flag if applicable:

Out of stock items are marked as out of stock:

View Product Details
To view product details, click the "Product Information" button. On grid view this will be shown as a icon only.
