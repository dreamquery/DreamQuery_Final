Sale Gift Certificate

Overview
Here you can edit sale gift certificate information and / or resend the gift certificate info to the recipient.
Sender / Recipient Details
Edit details where applicable.

Note: For bigger textarea, click the crosshair, top right, in the textarea box for message.
Value / Redeemed
Adjust if required. This is the monetary value of the certificate and the amount redeemed.
Admin Notes
Backend admin notes, not seen by sender or recipient.

Note: For bigger textarea, click the crosshair, top right, in the textarea box.
Enabled
Enables or disables gift certificate
Resend to Recipient
Click to resend gift certificate details to recipient. Useful if email hasn`t been received after purchase. Or if you need to change the email and resend etc


Note that if you have changed the recipient email address, click "Update" to update information in database first.
