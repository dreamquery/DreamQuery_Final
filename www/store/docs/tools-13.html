Batch Update Stock from CSV

Overview
For quickness you can batch update your stock levels from a csv import. The file is expected to have 2 columns containing the following comma delimited data:

Product Code,New Stock Level

If a product code isn`t recognised it is ignored. If more than one product has the same code, ALL will be updated.

You can have as many products in the CSV as you need. Header rows will be ignored if present.

It is recommended you back up your database before batch operations in case you need to revert back.

Finally, you can also batcn update selected stock levels from the stock overview screen.
