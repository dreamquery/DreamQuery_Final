Product - Manage Pictures

Overview
Here you can add product pictures. Pictures can be uploaded directly to your server or linked to from another location, eg: dropbox
Standard Pictures
Local image uploaded directly to your server. Use the and buttons to add more upload fields.

Supported local image formats: JPG / JPEG / PNG / GIF

IMPORTANT! Make sure your images are optimised for the web. Uploading large files will impact on your store load time. 800 x 640 @ 90 dpi with file size no more than 150KB is recommended for best performance A number of free image resizing utilities exist, so search on Google.
In the store front, the picture full size previews are controlled by SwipeBox.
Save Images to Folders
Use the option provided to create new picture folders if required. Click the button to add new folders. Use the dropdown menu to select the destination folder for the images. When entering name, only the following characters are allowed:

0 to 9, A to Z, - & _
(Letters, numbers, hyphens and underscores)
Folders added must be removed manually.
Remote Images
Can be batch added. The CSV expects three columns. NO header column should be sent. Columns are:

Image = Full http(s) path to remote image.
Thumb = Full http(s) path to remote thumb if applicable. Blank column if no thumb.
Display = yes or no to determine main display image. If not specified first image will be main display image. Example:

http://www.example.com/image1.jpg,http://www.example.com/tmb1.jpg,no
http://www.example.com/image2.jpg,http://www.example.com/tmb2.jpg,yes
http://www.example.com/image3.jpg,http://www.example.com/tmb3.jpg,no
If you are utilising SSL, your images MUST be loaded via https:

https://www.example.com/image1.jpg,https://www.example.com/tmb1.jpg,no
https://www.example.com/image2.jpg,https://www.example.com/tmb2.jpg,yes
https://www.example.com/image3.jpg,https://www.example.com/tmb3.jpg,no
IMPORTANT! Make sure hotlink protection isn`t enabled for the remote location. If you have hotlink protection enabled, you`ll need to add your store IP as an allowed IP. If you don`t whitelist you store your remote images may not load properly.
Cloud storage sites such as Google Drive and Dropbox can be useful for remote images.
Manage Pictures
Use the option to filter local / remote images and use the checkboxes to set the main display image. Click the to delete an image.
SEO Title / Alt Tags
Use the boxes provided to set alternative text for the image title and / or alt tags.

<img src="#" title="[TITLE]" alt="[ALT]">

For better SEO (Search Engine Optimisation). If you don`t understand this, leave the boxes blank.
