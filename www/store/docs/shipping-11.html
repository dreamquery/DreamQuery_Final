Shipping - Per Item Rates

Overview
Per item rates enable you set a shipping rate for the first product and then a set price for each additional product purchased.
Select Zones
Select the zones new rate applies to.
First Product Shipping Rate
Enter shipping rate for first product in basket. eg: 9.99
Additional Product(s) Shipping Rate
Enter shipping rate for each additional product in basket. eg: 9.99
Enable Per Item Rate
Enable or disable this per item rate
Batch Enable/Disable
Click the icon to batch enable or disable rates. Click the link and select the zones applicable.
Manage Rates
Use the links provided to edit or delete rates. Use the icon to filter by enabled countries.
