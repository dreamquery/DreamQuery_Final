Maian Cart Overview - Currency Converter

Currency Converter
This enables visitors to see an estimated value in their own currency. Note the processing is always in your default payment currency as set in the settings and not in the currency selected by a visitor.
Currency Selection
Selection is made via the drop down menu at the top of the screen. This shows currency options. Click button again to close with no selection. The current selected currency is shown with a icon. Select currency to reload screen. See following preview:


If a visitor is logged in, currency setting is remembered for next time.
