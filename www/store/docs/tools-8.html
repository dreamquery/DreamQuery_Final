Newsletter/Mailer

Overview
Accounts have an option to sign up for the newsletter and this enables batch sending mail to relevant accounts.

If you have a lot of subscribers, it is recommended you use a more robust mailing system.
Top Bar Options
- Filter accounts by type

- Send batch email. More info here. If you have used the filter the batch mail screen will mirror the filter. So, if you select trade only accounts, the mailer will only send to trade accounts.

- Save list to CSV
Edit / Delete
Use the relevant icons to edit an account or remove it from the newsletter list. Note that clicking does NOT delete an account, it simply disabled the newsletter for that account.
