Marketing Tracker

Overview
This is a simple feature to enable you to see where traffic to your site is coming from. A tracker is simply a code appended to the url. You can use different trackers in different campaigns to see if they are generating traffic. Cookies aren`t used with marketing trackers.
New Tracker
Enter code and name for new tracker. Use the option to auto generate a tracking code. A usage example is shown beneath the box. If you prefer to use a different variable than the default "tr", you can change it in the following file:

control/defined.php
Set the 'TRACKING_CODE_PREFIX' to your own preference. Example:

define('TRACKING_CODE_PREFIX', 'trackcode');
You would then use:

http://localhost/DEV/new/maian_cart/store/?trackcode=1234567
Manage Trackers
Order filters via the icon. Revenue and clicks are shown. Use the checkboxes to reset trackers and the edit and delete options to update or remove.
