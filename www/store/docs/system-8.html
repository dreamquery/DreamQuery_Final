Homepage Blog

Overview
This is a basic homepage blog system that enables you to post custom messages on your store homescreen. The messages appear only on the homescreen. Messages are not paginated, so you are recommended not to have too many messages on the homescreen. Note also that if you have a custom page set as the landing page, the blog will never been seen.
Title / Message
Enter title and message for blog entry. Use BB Code for formatting in the message body.

Note: For bigger textarea, click the crosshair, top right, in the textarea box.
Publish Date
This is the date the blog entry will appear. Can be set in the future if required. Only dates that match elapsed dates or the current date will display providing the entry is enabled.
Auto Delete Date
This is the date the blog entry will be removed from the system. Leave blank for no auto deletion.
Creation Date / Enable
Shown only on the edit screen. The creation date is only for backend reference. If a post is disabled it will NOT display at all.
Edit / Delete
Click the icon to edit an entry and the icon to delete an entry.
