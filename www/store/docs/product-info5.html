Product - Product MP3

Overview
Here you can add product MP3 files. Useful if you are selling music related products.
Add MP3 Files
Unlimited mp3 previews are supported. To add MP3 files, upload .mp3 files into the 'content/mp3/' folder. You can upload .mp3 files into this root folder or include them inside sub folders. Examples of structure:

content/mp3/file1.mp3
content/mp3/file2.mp3

Inside sub folders:

content/mp3/my-music/file1.mp3
content/mp3/folder/file2.mp3

Once mp3 files are in place, refresh MP3 admin screen. Click the icon to show the drop down menu to load folders.

Use the checkboxes to add mp3 files. You may assign mp3 files to a product from different mp3 folders. Enter a mp3 name/title if applicable. This is optional.

Finally, click the icon to play music file. Only one file can be played at a time.
Manage MP3 Files
You can change the display order by dragging / dropping the track rows. Use the icon to play a file, the icon to edit a file and finally, the icon to delete a file. Note that deleting a preview won`t delete the file from the server.
