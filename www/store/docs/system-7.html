Global Search

Overview
This screen is loaded via any search keyword entered on the main admin screen in the quick search box. This box is only shown to the global user and if a staff member has access to one or all of the sections searched.
Search Areas
The following searches are performed and results are shown in tabs:

Sales, Products, Accounts, Staff
Filters / Batch Ops
No filters or batch ops are available on this screen, if you wish to filter / batch process results use the search options in the relevant section.
Delete Options
No delete options are available on this screen, if you wish to delete results use the search options in the relevant section.
