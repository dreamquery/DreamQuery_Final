View Sales - Batch Update

Overview
In batch mode you can apply a status to multiple sales. Useful if you process in bulk. For attachments you should use the individual option as they are NOT available in batch mode. Most of the options here are the same as the standard update screen.
E-Mail Tags
The batch mode has more tags you can utilise. The following tags can be included anywhere in the message and subject (accept for {ORDER}):

{ORDER-NO} = Order No
{ORDER} - Product Order Info. Displays products ordered and prices.
{DOWNLOADS} - Download Page Link. Displays link to download page if downloads have been ordered.
{DATE} - Order Date. Displays order date.
{NAME} - Buyer Name. Useful for email personalisation.
Select Sales
Use the checkboxes next to each sale to determine which sale(s) this update applies to.
