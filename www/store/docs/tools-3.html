Move Products

Overview
This handy feature enables you to batch move products between categories. You can specify only certain products if you wish. Useful if you want to set up a special offer category and need to move products to this category.
Source Category
Source is the category where the products currently reside. On selecting a source category, products load in that category.
Destination Category
Destination is the category you would like to move the products to. Use the checkboxes to select the products you would like to move.
