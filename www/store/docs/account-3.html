Manage Trade Accounts

Overview
Here you can manage trade accounts only.
Top Bar Options
- Click to add new account

- Click to filter list view

- Click to view search box to search accounts

- Click to export accounts to CSV. Follows filters and search if set.
Account Specific Options
- Click to edit account

- Click to view sales for this account

- Click to add / edit backend notes

- Click to expand area for more options (below).
Expanded Area Options
Displays account information, plus the following actions:

Enable / Disable - Click to enable or disable account. Launches popup window to enter reason account is disabled (if applicable). If changing status, screen may need refresh to see status change.

Restricted Categories - If set, click "Yes" to view category restrictions in popup window.

Account Area Message - Enables you to post a message to the account area which will be seen on login. You can set an expiry date for the message if applicable.
Delete
Use the checkboxes to delete accounts.
