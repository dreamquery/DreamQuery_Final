Gift Certificates Overview

Overview
Enables you see purchased gift certificates. Gift certificates can be edited, deleted or disabled if required.
Filters
Use the drop down filter by clicking to show all certificates, certificates not redeemed or certificates disabled.
Search - Keywords or Date
Click to show the search box. The search box searches the following fields:

From Name, From Email, To Name, To Email, Code, Notes or Date Added.

For date search enter date in MySQL/US date format, which is the native storage format.

For example, to search for all certificates purchased on April 17 2013, enter 2013-04-17
Export
Overview can be exported to CSV if required by clicking the icon. If filters are used, export will mirror filters.
Edit / Delete
This lets you edit, delete or disable a gift certificate.
