Other Payment Options

Overview
Besides the payment processing options, Maian Cart also supports static options. These are: Bank Transfer, Phone Order, Payment by Cheque/Check & Cash on Delivery. For any of these options, futher communication between the vendor and the buyer is required. After using one of these methods, further instructions should be presented to the buyer.

Also, supported is On Account, which will treat the order as completed and perform all normal operations. This should ideally ONLY be used for trade accounts.

To enable any of these options, select 'Sales > Payment Methods'.

For more information, choose an option below:
