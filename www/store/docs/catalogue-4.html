Brands
A handy function is to group products into brands. For example, you may have a clothes store and want to group products by brand. Example: Berghaus, North Face, Craghoppers. This feature enables site visitors to narrow the category down to only manufacturing brands they are interested in.

Brands can be category specific. Useful if brands are applicable to more than a single category. As with categories, brands can be enabled/disabled.

Batch Add Brands
If you have a number of brand names, the batch option may be useful. File must have a single entry per line and not exceed 250 characters. If a batch file is uploaded and a name is entered, the name is ignored in favour of the file entries.
All Categories
If a brand is added to 'All Categories' it appears in all brand filters for categories. Brands added via this option will be shown next to all categories in admin and be prefixed "(ALL)". Deleting one of these brands removes it from all categories.
Manage Brands
Use the icon to filter brands by category. Use the checkboxes to delete and click a brand to edit it.
