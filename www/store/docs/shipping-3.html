Shipping Services

Overview
Weight based shipping services are services offered in any particular zone. For example, if you are a UK seller you might have services such as 'First Class Royal Mail', 'Second Class Royal Mail', 'Standard Parcels' etc. For each service, enter the estimated delivery time and whether this option requires a signature.

Weight based shipping services are not required.
Service Name
Enter new service name. UK examples:

First Class Royal Mail
Second Class Royal Mail
Standard Parcels
International Signed For
City Link Parcel Courier
Estimated Delivery
This is the estimated delivery time. Examples:

1 Day
3 Days
Next Day Delivery
Signature Required
Specify whether a signature is required on delivery for this service. If you aren`t sure, check with your mail provider.
Enable COD
If you have the Cash on Delivery (COD) option enabled as a payment method, can it be selected if this service is selected?
Apply to Zone
Specify the zone(s) this services applies to. Use the checkboxes to batch add service to multiple zones if in add mode. If in edit mode, use select option.
Batch Adding Services via CSV
Use the locate CSV option to batch add services. To batch add, csv must be a comma delimited list in the following format:

Service Name, Estimated Delivery, Signature Required (yes or no ONLY), Enable COD (yes or no ONLY)

Example:

Service1, 3 Days, yes, no
Service2, 4 Days, no, yes

For batch operation, file and checkbox selection of applicable zones is required.
Manage Shipping Services
Use the links provided to edit or delete services. Use the icon to filter by zone.
