Cart Online/Offline

Cart Online/Offline:
This feature lets you easily disable the cart system. For example, if site maintenance is being performed or if you are on holiday and cannot honour order delivery times. If you disable the cart, enter a reason for disabling the cart. You can also set a date to auto enable the cart.

If the date field is left blank, cart is permanently disabled until you enable it.
Enable Cart
Enable/disable cart. Useful for site maintenance.
Offline Reason
Enter reason for cart being offline. Use BB Code for formatting, HTML is not allowed.

Note: For bigger textarea, click the crosshair, top right, in the textarea box.
Date to Enable
If you wish to enable the cart on a certain date, specify it here. Leave blank to leave the cart disabled permanently.
If Disabled, Allow for IP Address(es)
This feature is useful if you don`t want to open your store to the public, but you want to view the store yourself. Useful for updates and changes. Enter multiple IP addresses if required separated with a comma. IPs entered can access the store when disabled.
