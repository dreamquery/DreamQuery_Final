Maian Cart Overview - Checkout

Overview
The Maian Cart checkout routines are one of the more complex areas of the system. The initial screen concentrates on the products in your basket where as later screens relate to delivery and other options if applicable. Please see below for an overview of each area.
Basket Product Options
For each product, use the and icons to increase or decrease quantity. And use the icon to delete from basket.


Clicking the 'Clear All' link at the top of the cart will clear all items.


If basket items have personalisation, an option will be shown to edit the details. Click the edit link to edit if required.


If basket items have gift certificates, an option will be shown to edit the details. Click the edit link to edit if required.

Checkout - Account or Guest
With Maian Cart you can check out by logging into your account, or by checking out as a 'Guest'. If you have an account, this will have benefits as certain cart data will be pre filled, such as delivery addresses. For 'Guest' checkout, you`ll need to enter your payment information every time.


If 'Guest' is chosen, an option to create an account will appear during checkout. More info below.
Return to Basket
Once the option to log in or checkout as guest has been selected a new screen loads with other checkout options. To return to the basket you can click the icon or the 'Update / View Basket' button in the left hand menu.

Checkout Options
A drop down appears with checkout options.


You can jump to specific areas of the checkout system, but if a previous option is required you will return to that screen. For example, if you choose the final screen and a delivery / billing address is required you will be taken back to those screens. Therefore it is recommended you use the 'Continue' buttons to proceed through the screens.
Checkout Options - Billing Address
Are required. If you have an account, these will be auto filled in via the billing information you set in your account profile.
Checkout Options - Delivery Address
Same as billing, but may not be shown if no delivery options are needed. You can use the icon to copy data from billing to delivery fields with one click.


Note that for delivery options, a phone number may also be required.
Checkout Options - Shipping Cost
On this shipping cost screen, select your preferred zone via "Please choose your preferred shipping region/zone" and choose a shipping option.

Checkout Options - Coupon Code
Enter coupon code if applicable. To remove the code, return to the coupon screen, clear box, then click continue.
Checkout Options - Account
If guest checkout was select, this screen appears asking if buyer wants to create account. This is entirely optional.

Checkout Options - Notes
Option for visitor to add order notes.
Checkout Options - Pay & Finish - Payment Method
Choose preferred payment method. Clicking the 'Payment Method Info' link opens a pop up div with details about the relevant payment method.

Checkout Options - Pay & Finish - Insurance
Insurance will only be shown if enabled. Click the icon to view more information about the insurance.


Click the link to add or remove insurance.

Checkout Options - Pay & Finish - Terms & Conditions
Terms and conditions are only shown if enabled. If enabled, buyer MUST click the checkbox to acknowledge the T&C, which are viewable by clicking the 'Terms & Conditions' text, which opens a small div window.

Checkout Options - Pay & Finish - Complete Order
Click to complete order. If none payment gateway method, screen will be shown with further instructions.
