Shipping Zones

Overview
Groups areas or districts in a country into a zone. 'Zone' is just a generic term applied to this page, your country doesn`t specifically need to support zones. For example, for UK zones you may want to include certain counties into one zone and other counties into another, depending on location. If no zones are specified, shipping selectors on checkout page are disabled in favour of a simplified checkout screen with no shipping options.

If you aren`t bothered about zones, simply create a single zone and call it 'All Areas' for all countries.
Name of Zone
This is the name of your zone. Its not used in the store and just acts as a reference in the admin area.
Specify Countries
Which countries does this zone apply to.
Enter Zone Areas/Regions
Enter zone areas/regions. One per line. UK example might be Counties:

Derbyshire
Staffordshire
West Midlands

OR use batch upload option. If you enter text in the box AND batch upload, the text box is ignored in favour of the upload file.
Or Locate Batch File
Batch add entries via a text file. ONE entry per line. If this option is utlised, the text box is ignored, regardless of data.
Tax Rate
Does a tax rate apply to this zone? Enter 0 for no tax.
Apply Tax to Shipping
If a tax rate is specified, should this rate also be applied to the shipping? If yes, total is total+shipping+rate. If no, total is total+rate+shipping.
No Shipping at All
If shipping to this country is free, enter a tax rate of 0 and specify NO tax on shipping. Give the zone a generic name like 'All Zones'.
Manage Zones
Use the links provided to edit or delete zones. Use the icon to filter by enabled countries.
