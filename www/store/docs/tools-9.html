Low Stock Export

Overview
Enables you to export low stock items to CSV. Useful if you need to place orders with suppliers.
Categories
Select category or categories from which products/attributes will be exported.
Where Stock Levels Between
Specify the stock level range for export. For example, if you only wanted to export products/attributes that had less than 5 as a stock level, you would enter 0 & 5.
Include Disabled Items
If you have items disabled, do you also want to include these in the export?
