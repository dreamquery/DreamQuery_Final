Shipping - Weight Rates

Overview
Weight based shipping rates are the rates you offer for products in a certain weight band for a certain service. This can be any kind of weight band, but must match your product weight. For example, if your products are in grammes, use a gram range. eg: 0 to 1000. Weight isn`t shown in the cart or on the checkout page.

Weight based shipping rates are not required.
Weight From / Weight To
Weight range. First rate should start 0. To specify only a single rate, enter a high amount for the 'from' field. This can be any kind of weight band, but must match your product weight. For example, if your products are in grammes, use a gramme range. eg: 0 to 1000. If pounds and ounces you might start 0 to 5.
Shipping Cost
Shipping cost for this rate.
Apply to Service
Which service(s) does this rate apply to? Use the checkboxes to batch add rates to multiple services if in add mode. If in edit mode, use select option.
Batch Adding Rates via CSV
Use the locate CSV option to batch add rates. To batch add, csv must be a comma delimited list in the following format:

Weight from, Weight to, Shipping Cost (no commas, etc, 9.99)

Example:

0, 10000, 1.99
100001, 500000, 5.99

For batch operation, file and checkbox selection of applicable services is required.
Manage Rates
Use the links provided to edit or delete rates. Use the icon to filter by service.
